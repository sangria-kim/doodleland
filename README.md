# 🎨 Drawing Sandbox — 아이 그림 인터랙티브 샌드박스 앱

아이(3~5세)가 종이에 그린 그림을 촬영하면 배경을 자동으로 제거하고, 테마 무대 위에서 움직이는 캐릭터로 만들어주는 인터랙티브 놀이 앱입니다.

<br>

## 핵심 기능

### 📷 그림 가져오기
카메라로 촬영하거나 갤러리에서 사진을 선택합니다. 자유 크롭으로 그림 영역을 지정하면 앱이 배경을 자동으로 제거하여 투명 PNG 캐릭터로 만들어줍니다.

### 🎪 무대 놀이
5종의 테마 배경(숲, 하늘, 바다, 별밤, 사탕나라) 위에 내가 만든 캐릭터를 올려놓고 놀 수 있습니다. 각 캐릭터에 움직임을 선택하면 둥실둥실 떠다니거나, 통통 튀거나, 빙글빙글 돌며 무대 위에서 살아 움직입니다.

### 👆 터치 반응
무대 위 캐릭터를 탭하면 통통 점프합니다. 드래그로 자유롭게 위치를 옮길 수 있고, 같은 캐릭터를 여러 개 올려 나만의 장면을 꾸밀 수 있습니다.


## v1 범위 및 제한

- 배경 제거는 규칙 기반만 사용 (AI/ML 미사용)
- 캐릭터 탭 반응은 공통 통통 점프 1종
- 무대 씬은 영속 저장하지 않음 (앱 재시작 시 초기화)
- 동시 배치 캐릭터 최대 10개
- 한국어 전용 (i18n 미적용)
