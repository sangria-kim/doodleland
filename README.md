# 그림놀이터

아이(3~5세)가 그린 그림을 촬영하거나 고른 뒤 잘라내고, 저장 시 규칙 기반으로 배경을 제거해 캐릭터로 보관한 다음 무대 위에서 움직여 볼 수 있는 Flutter 앱입니다.

## 현재 구현된 핵심 기능

### 그림 가져오기
- 카메라 촬영 또는 갤러리 선택을 지원합니다.
- 네이티브 `image_cropper` 화면으로 이동해 자유 비율로 크롭합니다.
- 저장 버튼을 누를 때 원본 복사, 배경 제거, 썸네일 생성, DB 저장까지 한 번에 처리합니다.

### 그림 라이브러리
- 저장된 캐릭터 메타데이터는 Drift(SQLite)에 저장합니다.
- 원본, 투명 PNG, 썸네일은 앱 내부 저장소에 별도 보관합니다.
- 라이브러리 카드 길게 누르기로 캐릭터와 관련 파일을 함께 삭제할 수 있습니다.

### 무대 놀이
- 5종 배경(숲, 푸른 하늘, 바다, 별빛 밤, 캔디 랜드)을 고를 수 있습니다.
- 캐릭터 추가 시 5종 모션(둥실둥실, 통통 점프, 씽씽 활공, 데굴데굴, 빙글빙글) 중 하나를 선택합니다.
- 무대 위 캐릭터는 탭 반응, 드래그 이동, 길게 눌러 제거를 지원합니다.
- Android에서는 immersive fullscreen으로 동작합니다.

## 현재 동작 기준 제약 사항

- 미리보기 화면은 현재 크롭 결과를 보여주며, 투명 배경 결과는 저장 시 생성됩니다.
- 캐릭터 이름은 현재 시간 기반 문자열로 자동 생성되며, 이름 수정 기능은 없습니다.
- 무대 씬은 영속 저장하지 않으며 앱 재시작 시 초기화됩니다.
- 동시 배치 캐릭터 수는 최대 10개입니다.
- AI/ML 기반 분류나 모션 추천은 포함하지 않습니다.

## 타깃 플랫폼

- iOS
