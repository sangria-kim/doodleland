# 04. 화면 흐름 & UI 가이드

> **MVP(v1) 기준 메모:** 배경 제거는 앱 내에서 자동 처리한다. AI 관련 화면이나 설정 단계는 없다. 무대에서 캐릭터를 탭하면 모든 그림에 공통으로 적용되는 **통통 점프**가 재생된다.

## 전체 화면 구성

| # | 화면 | 경로 | 설명 |
|---|------|------|------|
| 1 | 홈 | `/` | 진입점, 두 가지 메인 액션 |
| 2 | 캡처 - 소스 선택 | `/capture` | 카메라/갤러리 선택 |
| 3 | 캡처 - 크롭 | `/capture/crop` | 그림 영역 지정 |
| 4 | 캡처 - 미리보기 | `/capture/preview` | 배경제거 결과 확인 |
| 5 | 배경 선택 | `/stage/background` | 놀이 배경 선택 |
| 6 | 무대 | `/stage` | 메인 놀이 화면 |

> **참고:** 라이브러리는 독립 화면이 아니라 무대 화면의 캐릭터 추가 바텀시트로만 접근한다.

## 네비게이션 흐름

```
[홈]
  ├─ "그림 만들기" → [소스 선택] → [크롭] → [미리보기]
  │                                           ├─ "저장하기" → [홈] (완료 토스트)
  │                                           ├─ "저장하고 하나 더!" → [소스 선택] (완료 토스트)
  │                                           └─ "다시 찍기" → [소스 선택]
  │
  └─ "놀이 시작" ─┬─ (캐릭터 0개) → 토스트 안내 + [소스 선택]으로 이동
                  └─ (캐릭터 1개+) → [배경 선택] → [무대]
                                                     │
                                                     ├─ "+" 버튼 → 캐릭터 선택 바텀시트
                                                     │              → 움직임 프리셋 선택
                                                     │              → "등장하기" → 캐릭터 배치
                                                     │
                                                     ├─ 🎨 배경 변경 버튼 → [배경 선택]
                                                     │   (배경만 교체, 배치된 캐릭터 유지)
                                                     │
                                                     ├─ 캐릭터 탭 → 공통 통통 점프 재생
                                                     ├─ 캐릭터 드래그 → 위치 이동
                                                     ├─ 캐릭터 길게 누르기 → 제거 옵션
                                                     └─ "←" 뒤로가기 → [홈]
```

## 화면별 상세

### 1. 홈 화면

```
┌─────────────────────────────────────────────┐
│                                             │
│          [앱 로고 / 타이틀 일러스트]           │
│                                             │
│    ┌──────────────┐   ┌──────────────┐      │
│    │  🎨 그림      │   │  🎪 놀이     │      │
│    │   만들기      │   │   시작       │      │
│    └──────────────┘   └──────────────┘      │
│                                             │
└─────────────────────────────────────────────┘
```

- 가로 모드 전체 화면
- 큰 버튼 2개 (최소 120×120dp)
- 아이콘 + 짧은 텍스트
- 배경: 부드러운 파스텔 색상 또는 간단한 일러스트
- **놀이 시작 가드:** 캐릭터가 0개일 때 `놀이 시작`을 누르면 "먼저 그림을 만들어보세요!" 토스트 표시 후 캡처 화면으로 이동

### 2. 캡처 - 소스 선택

```
┌─────────────────────────────────────────────┐
│ ←                                           │
│                                             │
│    ┌──────────────┐   ┌──────────────┐      │
│    │  📷 카메라    │   │  🖼️ 갤러리   │      │
│    │              │   │              │      │
│    └──────────────┘   └──────────────┘      │
│                                             │
└─────────────────────────────────────────────┘
```

- 카메라/갤러리 큰 버튼 2개
- 선택 즉시 `image_picker` 호출
- 이미지 획득 후 자동으로 크롭 화면 이동

### 3. 캡처 - 크롭

```
┌─────────────────────────────────────────────┐
│ ←                            [ ✂️ 자르기 ]  │
│                                             │
│   ┌───────────────────────────────┐         │
│   │                               │         │
│   │     ┌ ─ ─ ─ ─ ─ ─ ┐          │         │
│   │     │  크롭 영역    │          │         │
│   │     │  (드래그 조정) │          │         │
│   │     └ ─ ─ ─ ─ ─ ─ ┘          │         │
│   │                               │         │
│   └───────────────────────────────┘         │
│                                             │
└─────────────────────────────────────────────┘
```

- 자유 직사각 크롭 UI (`image_cropper` 네이티브 UI)
- 자동 감지된 추천 박스가 기본 선택
- `자르기` 버튼으로 확정

### 4. 캡처 - 미리보기

```
┌─────────────────────────────────────────────┐
│ ←                                           │
│                                             │
│   ┌───────────────────────────────┐         │
│   │  ░░░░░░░░░░░░░░░░░░░░░░░░░░ │         │
│   │  ░░░░ [캐릭터 이미지] ░░░░░░ │         │
│   │  ░░░░░░░░░░░░░░░░░░░░░░░░░░ │         │
│   └───────────────────────────────┘         │
│           (체크무늬 = 투명 영역)              │
│                                             │
│  ┌──────────┐ ┌──────────┐ ┌────────────┐  │
│  │🔄다시찍기│ │💾저장하기│ │➕하나 더!  │  │
│  └──────────┘ └──────────┘ └────────────┘  │
│                                             │
└─────────────────────────────────────────────┘
```

- 배경제거 결과를 체크무늬 배경 위에 표시
- 배경 제거는 앱 내 로직으로 자동 처리
- 처리 중 로딩 인디케이터 표시 (스피너 + "그림을 꾸미는 중이에요!")
- **품질 경고:** 배경 제거 결과가 불량하면 화면 상단에 안내 메시지 표시 (저장은 허용)
- `저장하기` → 라이브러리 저장 후 홈으로 이동 (완료 토스트)
- `저장하고 하나 더!` → 라이브러리 저장 후 소스 선택으로 이동 (완료 토스트)
- `다시 찍기` → 소스 선택으로 돌아감

### 5. 배경 선택

```
┌─────────────────────────────────────────────┐
│ ←          배경을 골라봐!                     │
│                                             │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐        │
│  │ 숲 │ │하늘│ │바다│ │별밤│ │사탕│        │
│  │    │ │    │ │    │ │    │ │나라│        │
│  └────┘ └────┘ └────┘ └────┘ └────┘        │
│                                             │
│       ← 가로 스크롤 →                        │
│                                             │
└─────────────────────────────────────────────┘
```

- 가로 스크롤 카드 목록
- 카드 크기: 최소 160×120dp
- 배경 썸네일 + 이름
- 탭하면 즉시 무대 화면으로 이동

### 6. 무대 (메인 놀이 화면)

```
┌─────────────────────────────────────────────┐
│ [←]                              [🎨 배경]  │
│                                             │
│              🌳🌳        ☁️                  │
│                                             │
│         [캐릭터A]    [캐릭터B]               │
│                  [캐릭터C]                   │
│  ─────────────── 바닥선 ──────────────────   │
│                                             │
│                              [ ＋ 추가 ]     │
└─────────────────────────────────────────────┘
```

- **배경:** 선택된 배경이 전체 화면을 채움
- **캐릭터:** 배경 위에 자유롭게 배치, 각자 선택된 기본 움직임을 수행
- **탭 반응:** 모든 캐릭터는 탭 시 공통 `통통 점프`를 재생
- **하단 우측 `+` 버튼:** 캐릭터 추가 (바텀시트 열림). 10개 도달 시 비활성화 + "무대가 꽉 찼어요!" 툴팁
- **우상단 `🎨 배경` 버튼:** 배경 선택 화면으로 이동. 배경만 교체되고 배치된 캐릭터와 위치는 유지됨
- **좌상단 `←`:** 홈으로 돌아감
- **UI 최소화:** 놀이 중에는 버튼을 최대한 숨김 (일정 시간 터치 없으면 버튼 페이드아웃 검토)

### 캐릭터 추가 바텀시트

```
┌─────────────────────────────────────────────┐
│  그림을 골라봐!                               │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐               │
│  │그림│ │그림│ │그림│ │그림│               │
│  │ 1  │ │ 2  │ │ 3  │ │ 4  │               │
│  └────┘ └────┘ └────┘ └────┘               │
│                                             │
│  움직임:  [둥실] [통통] [씽씽] [데굴] [빙글]  │
│  탭 반응: 공통 통통 점프 (자동 적용)          │
│                                             │
│            [ 🎉 등장하기! ]                   │
└─────────────────────────────────────────────┘
```

- 1단계: 캐릭터 선택 (가로 스크롤 썸네일)
- 2단계: 움직임 프리셋 선택 (칩/토글 버튼)
- 3단계: 탭 반응은 별도 선택 없이 공통 기본 동작 자동 적용
- `등장하기!` 버튼으로 무대에 배치
- 한 화면에 필요한 선택을 모두 완료할 수 있도록 구성

## 공통 UI 규칙

### 레이아웃
- 전체 앱 가로(landscape) 고정
- SafeArea 적용 (노치/펀치홀 대응)
- 태블릿 대응: 최대 너비 제한 없이 자연 확장

### 터치 영역
- 모든 인터랙티브 요소 최소 **56×56dp** (3~5세 대상)
- 주요 버튼은 **80dp 이상** 권장

### 텍스트
- 최소 폰트 크기: 18sp
- 텍스트 최소화, 가능한 아이콘/일러스트로 대체
- 한국어 기본, 텍스트 하드코딩 (MVP에서는 i18n 불필요)

### 색상 (참고)
- 밝고 따뜻한 파스텔톤 기본
- 배경 선택/프리셋 선택 시 선택 상태는 강조색으로 표시
- 구체적인 색상 팔레트는 디자인 단계에서 확정

### 애니메이션
- 화면 전환: 기본 슬라이드 (300ms)
- 바텀시트: 기본 스프링 애니메이션
- 모든 캐릭터 애니메이션: 60fps 유지 목표
