# 04. 화면 흐름 & UI 가이드

## 현재 라우트 구성

| 화면 | 경로 | 현재 역할 |
|------|------|-----------|
| 홈 | `/` | 시작 화면 |
| 그림 가져오기 | `/capture` | 카메라/갤러리 선택 |
| 이미지 크롭 | `/capture/crop` | 네이티브 크롭 진입 및 재실행 |
| 미리보기 | `/capture/preview` | 저장 전 크롭 결과 확인 |
| 배경 고르기 | `/stage/background` | 5종 배경 선택 |
| 무대 | `/stage` | 캐릭터 배치 및 상호작용 |

## 실제 네비게이션 흐름

```text
[홈]
  ├─ 그림 만들기
  │   └─ [/capture]
  │      └─ 이미지 선택
  │         └─ [/capture/crop]
  │            ├─ 크롭 취소 -> [/capture]
  │            └─ 크롭 완료 -> [/capture/preview]
  │               ├─ 저장하기 -> 저장 처리 후 [/]
  │               ├─ 저장하고 하나 더! -> 저장 처리 후 [/capture]
  │               └─ 다시 찍기 -> [/capture]
  │
  └─ 놀이 시작
      ├─ 캐릭터 없음 -> 스낵바 후 [/capture]
      └─ 캐릭터 있음 -> [/stage/background] -> [/stage]
                                         ├─ + 버튼 -> 캐릭터 선택 시트
                                         ├─ 배경 버튼 -> [/stage/background]
                                         └─ 뒤로가기 -> [/]
```

## 화면별 현재 구현

### 1. 홈
- 중앙 정렬된 타이틀 `그림놀이터`
- 설명 문구 1줄
- 세로로 쌓인 큰 버튼 2개:
  - `그림 만들기`
  - `놀이 시작`
- 라이브러리 비어 있을 때 `놀이 시작`:
  - 스낵바 표시
  - 자동으로 `/capture` 이동

### 2. 그림 가져오기
- `SafeArea` 내부 단일 컬럼
- 타이틀: `그림 가져오기`
- 필요 시 상단 피드백 배너 노출
- 큰 버튼 2개:
  - `카메라로 찍기`
  - `갤러리에서 선택`
- 처리 중 하단 `LinearProgressIndicator`

### 3. 이미지 크롭
- AppBar 제목: `이미지 크롭`
- 진입 즉시 네이티브 크롭 UI 호출
- 화면 자체는 다음 역할을 겸함:
  - 원본 이미지가 남아 있을 때 임시 미리보기
  - 크롭 재실행 버튼 제공
  - 오류/취소 후 복구용 fallback
- 하단 버튼:
  - `크롭 다시 실행`

### 4. 미리보기
- AppBar 제목: `미리보기`
- 현재는 **투명 결과가 아니라 크롭 이미지를 보여줌**
- 하단 액션 3개:
  - `저장하기`
  - `저장하고 하나 더!`
  - `다시 찍기`
- 저장 중 하단 `LinearProgressIndicator`
- 저장 완료 후 스낵바:
  - 저장된 `characterId`
  - 배경 제거 품질 경고가 있으면 함께 표시

### 5. 배경 고르기
- AppBar 제목: `배경 고르기`
- 2열 그리드 카드
- 카드 내용:
  - 배경 이미지
  - 하단 반투명 라벨
- 탭 즉시 무대 이동

### 6. 무대
- AppBar 없는 전체화면 배경 무대
- 상단 오버레이:
  - 좌측 뒤로가기 아이콘
  - 우측 배경 변경 아이콘
  - 배치 수 텍스트 `n / 10`
  - 오류 카드
- 우하단:
  - `+` floating action button
  - 마지막 상호작용 후 2초 뒤 자동 fade-out
- 빈 상태:
  - 중앙 안내 문구

## 캐릭터 선택 바텀시트

### 1단계. 그림 선택
- 라이브러리 캐릭터를 반응형 그리드로 표시
- 카드 탭 시 2단계로 전환
- 빈 상태면:
  - 안내 문구
  - `그림 만들기` 버튼

### 2단계. 움직임 선택
- 좌측 또는 상단:
  - 선택한 캐릭터 미리보기
  - 현재 선택한 모션 라벨/설명
- 우측 또는 하단:
  - 모션 카드 목록
- 하단 액션:
  - 뒤로가기
  - 등장하기

## 무대 상호작용 규칙

### 탭
- 캐릭터를 앞으로 올림
- scale bounce 재생
- 재생 중 추가 탭 무시

### 드래그
- 위치 갱신
- 드래그 동안 기본 모션 정지
- 종료 후 기본 모션 재개

### 길게 누르기
- 즉시 무대에서 제거
- 결과를 스낵바로 표시

## 현재 문서상 주의사항
- 예전 문서에 있던 체크무늬 투명 미리보기, 자동 크롭 박스 추천, 가로 스크롤 배경 선택 UI는 현재 코드와 다릅니다.
- 현재 무대 렌더링은 `CustomPainter`가 아니라 개별 위젯 오브젝트를 `Stack`으로 배치하는 구조입니다.
